# AI Clinic Call Assistant

Monorepo for the AI Clinic Call Assistant & Task Automation Platform.

## Structure
- `backend/` FastAPI API service
- `frontend/` Web dashboard
- `docs/` Product and technical documentation
- `infra/` Infrastructure notes and deployment artifacts

## Quickstart (dev)
This repo is scaffold-only for now. Once we add dependencies, we'll provide:
- `backend/` local API run instructions
- `frontend/` local UI run instructions
